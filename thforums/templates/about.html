About Tavernhold

Tavernhold is a web forum for adventurers and mercenaries to connect, share stories, and embark on quests together.

In Fulfillment for the Project for
CPE 202 - Database Management System
Presented to:
Engr. Jayson Carl Ferrer

Technologies Used

Flask
Python web framework for backend logic and routing
Jinja2
Templating engine for dynamic HTML rendering
Bootstrap 5
Responsive front-end framework for layout and components
HTML5 & CSS3
Markup and styling for all pages
JavaScript
Interactive features and AJAX (commend system, sound effects)
Font Awesome
Icon library for UI elements
SQLite
Database for storing users, threads, and posts
WTForms
Form handling and validation
Werkzeug
Security and utility functions for Flask
Custom CSS
Custom-made Medieval-inspired theme and effects
