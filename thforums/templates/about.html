Technologies Used

  • Flask – Python web framework for backend logic and routing
  • Jinja2 – Templating engine for dynamic HTML rendering
  • Bootstrap 5 – Responsive front-end framework for layout and components
  • HTML5 & CSS3 – Markup and styling for all pages
  • JavaScript – Interactive features and AJAX (commend system, sound effects)
  • Font Awesome – Icon library for UI elements
  • SQLite – Database for storing users, threads, and posts
  • WTForms – Form handling and validation
  • Werkzeug – Security and utility functions for Flask
  • Custom CSS – Custom-made Medieval-inspired theme and effects
These technologies work together to provide a dynamic, interactive, and visually engaging forum experience.
